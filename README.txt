jav's WordPress post to Trello card  . will connect WordPress with trello .

== Description ==

`WordPress post to trello card`  will Create a Trello Card with your new Post ,or if there is a  POST Status Changed.

== HOW IT WORKS ==

It uses Trello API with your Trello API key. When there is a New POST or an post Status Change WordPress Hook initiate, By using those Hook It gets the Post Information and sends it to the Trello.

== Standard Features ==

* Easy-to-use, no unnecessary Setup Wizard for perfect configuration.
* No hidden menu or option, everything is crystal clear. 
* Show trello Boards and Lists.
* one click authentication.
* Create Trello Card after every new Post or create Trello card after certain  Post status Change.

== Installation ==

1. Upload `wordpress post to trello card` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Get Trello access Key and Past it . 
4. Select a Trello Board and list Where you want to create Cards.
5. Change the Settings by your Preference .enjoy
